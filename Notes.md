
# Environment setup
- PHP
  > brew install php
  > brew link php
- Composer
- Laravel
  > composer global require laravel/installer
  > Put ~/.composer/vendor/bin directory in your PATH
- Laravel valet
  > valet install
- PhpStorm
- Postman
- NodeJs
- DBngin
